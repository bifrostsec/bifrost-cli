# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in `bifrost-cli`, please report it privately by emailing [security@bifrostsec.com](mailto:security@bifrostsec.com).

Please include:

- A short description of the issue
- Steps to reproduce it
- The affected version or commit, if known
- Any suggested mitigation or fix, if available

Please do not open a public GitHub issue for security reports.

## What to Expect

We are a small team, so response times may vary, but we will review reports as quickly as we can.

If the report is confirmed, we will work on a fix and coordinate disclosure appropriately.
